## 3. Ideas para Implementar (Extraídas de Competencia)

### 3.1 De ProWritingAid

#### Reportes de Análisis Estilístico
| Reporte | Descripción | Prioridad | Complejidad |
|---------|-------------|-----------|-------------|
| **Sticky Sentences** | Detecta oraciones con exceso de "glue words" (el, y, que, de). Meta: <40% | ⭐⭐⭐ Alta | Media |
| **Pacing Check** | Identifica pasajes lentos por longitud párrafos y falta de diálogo | ⭐⭐⭐ Alta | Media |
| **Dialogue Tags** | Detecta exceso de "dijo/preguntó", sugiere reducir a <30% | ⭐⭐ Media | Baja |
| **Echo/Repetitions** | Palabras repetidas en proximidad, gráfico de longitud oraciones | ⭐⭐⭐ Alta | Baja |
| **Sensory Report** | Análisis de descripciones sensoriales (5 sentidos) | ⭐⭐ Media | Alta |
| **Sentence Variation** | Gráfico de distribución de longitud de oraciones | ⭐⭐ Media | Baja |

#### Funcionalidades IA (Sparks)
| Función | Descripción | Prioridad |
|---------|-------------|-----------|
| **Continue Writing** | Continúa la escena manteniendo estilo | ⭐⭐ Media |
| **Add Sensory Detail** | Enriquece descripción con detalles sensoriales | ⭐⭐ Media |
| **Expand/Condense** | Expandir o condensar pasajes | ⭐⭐ Media |
| **Change POV** | Convertir entre primera/tercera persona | ⭐ Baja |

### 3.2 De Novelium

#### Detección de Inconsistencias
| Función | Descripción | Prioridad | Complejidad |
|---------|-------------|-----------|-------------|
| **Timeline Conflicts** | Detecta secuencias imposibles de eventos | ⭐⭐⭐ Alta | Alta |
| **Character Consistency** | Alerta cambios en atributos físicos (ojos, pelo) | ⭐⭐⭐ Alta | Alta |
| **Knowledge Tracking** | Qué sabe cada personaje en cada momento | ⭐⭐⭐ Alta | Alta |
| **Character Location** | Detecta personaje en dos lugares simultáneos | ⭐⭐⭐ Alta | Media |
| **Deceased Character Alert** | Alerta si personaje muerto aparece después | ⭐⭐⭐ Alta | Media |

#### Visualización
| Función | Descripción | Prioridad |
|---------|-------------|-----------|
| **Timeline automático** | Construye timeline desde fechas en texto | ⭐⭐⭐ Alta |
| **Alertas en tiempo real** | Sidebar con inconsistencias mientras escribes | ⭐⭐ Media |

## 5. Oportunidades de Diferenciación

### 5.1 Nuestras Ventajas Únicas

| Ventaja | Descripción | Competidores con esto |
|---------|-------------|----------------------|
| **100% Offline** | Todo procesamiento local | Solo Scrivener (sin IA) |
| **LLM Local (Ollama)** | Privacidad total manuscritos | Ninguno |
| **Sistema Votación Multi-método** | Consenso entre múltiples análisis | Ninguno |
| **Español Nativo** | No es traducción, diseñado para español | Stilus, Lorca |
| **Tracking Conocimiento Personajes** | Qué sabe cada personaje | Novelium (cloud) |
| **Correferencias Avanzadas** | Resolución de menciones | Limitado en otros |

### 5.2 Gaps en el Mercado

| Gap | Descripción | Oportunidad |
|-----|-------------|-------------|
| **Español profesional offline** | No existe herramienta completa | ⭐⭐⭐ Alta |
| **Ficción + Privacidad** | Cloud es la norma, preocupa a 65% autores | ⭐⭐⭐ Alta |
| **Análisis narrativo + corrección** | Separados en mercado actual | ⭐⭐⭐ Alta |
| **Herramienta integral barata** | Competencia: $10-30/mes cada una | ⭐⭐ Media |

### 5.3 Mercado Español Desatendido

- ProWritingAid/Grammarly: Traducciones básicas, no nativas
- Novelium/Sudowrite: Solo inglés
- AutoCrit: Solo inglés
- Scrivener: Interfaz en español, sin análisis
- **Oportunidad**: Ser el "ProWritingAid para español" con análisis narrativo

## 6. Roadmap de Ideas Priorizado

### Fase 1: Quick Wins (1-2 meses)
1. **Sticky Sentences** - Detector de oraciones pesadas
2. **Echo Report** - Repeticiones en proximidad
3. **Sentence Variation Graph** - Visualización longitud oraciones
4. **Pacing Analysis básico** - Pasajes lentos por párrafos largos

### Fase 2: Diferenciadores (2-4 meses)
1. **Timeline automático desde fechas** - Extraer y visualizar
2. **Character Consistency Alerts** - Cambios de atributos
3. **Story Bible / Wiki de proyecto** - Entidades conectadas
4. **Export Scrivener** - Anotaciones preservadas

### Fase 3: Avanzado (4-6 meses)
1. **Knowledge Graph** - Conexiones semánticas
2. **Benchmarking por género** - Comparar con corpus
3. **Plantillas estructuras narrativas** - Save the Cat, etc.
4. **Stilus API integration** - Voter profesional español
